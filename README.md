## 프로젝트 소개

기억하고 싶은 사진 또는 책이나 영화를 메모와 함께 기억할 수 있게 해주는 웹사이트입니다. <br/><br/>
오늘 하루를 담은 사진, 책 표지 또는 영화 포스터 등의 이미지를 보여주고 자기만의 소감을 기록할 수 있습니다. <br/><br/>
회원가입, 로그인 등의 불편함을 지우기 위해 페이지의 local storage에 자료를 저장하고 불러오는 방식을 이용했습니다. <br/><br/>
React로 클라이언트를, Express.js로 서버를 구성했습니다. Heroku를 통해 배포하고 있습니다. <br/><br/>

## 시연

처음 사이트에 접속하게 되면 간단한 홈페이지 소개 글이 나옵니다.
📚 버튼이나 🎥 버튼을 클릭하면 각 공간으로 이동합니다. <br/><br/>
공간에서는 🔎 을 통해 검색어에 맞는 자료들을 찾을 수 있습니다.
원하는 자료를 클릭하면 해당 자료의 이미지 파일이 저장됩니다. <br/><br/>
📂 버튼을 클릭해 내 컴퓨터에서 이미지 파일을 직접 찾아 저장할 수도 있습니다.<br/><br/>
<img width='100%' src='https://user-images.githubusercontent.com/31424628/135713948-e2c51bae-58b3-479b-a735-977f31df0dee.gif' />
<br/><br/>
이미지를 클릭하면 두 개의 버튼이 나오는데 ✏️ 버튼과 🗑 버튼입니다. <br/><br/>
메모 버튼을 클릭하면 메모 창으로 넘어가게 되고 메모가 가능합니다. 메모 창 상단에는 ❎ 버튼과 ✅ 버튼이 있습니다. <br/><br/>
체크 버튼을 클릭하면 저장 완료 메세지가 뜨며 메모를 저장을 할 수 있고 저장 완료 메세지는 3초 후 사라집니다. <br/><br/>
취소 버튼을 클릭하면 원래 있던 창으로 돌아가게 됩니다. <br/><br/>
삭제 버튼을 클릭하면 경고 문구가 뜨고, 삭제를 진행하거나 취소할 수 있습니다.<br/><br/>
